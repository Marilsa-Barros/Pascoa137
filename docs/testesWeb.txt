Existem várias formas de organizar um projeto de teste web
- Simples (Se IDE / manualmente)
- Selenium + Cucumber (BDD)
- Page Objects

O Selenium usa um Driver para se comunicar com o browser

Código (Selenium)  ---> Driver ---> Browser

Existe um driver para cada versão ou conjunto de versões de um navegadro

No caso Chorme/Edge: 1 para cada versão
Nos demais, normalmente, 1 versão de driver funciona para várias do navegador

Há 3 forms de configurar/utilizar um driver:
- Download manual e instala dentro do projeto - Bom: Portátil - Ruim: Manutenção
- Download manual e instala em uma pasta compartilhada - Bom: Manutenção - Ruim: não é Portátil
- Gerenciador de download automático - Bom: Manutenção + Portátil - Ruim: Usa recursos
